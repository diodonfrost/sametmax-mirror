bash alias – Sam & Max http://sametmax.com Du code, du cul Thu, 05 Sep 2019 08:22:03 +0000 en-US hourly 1 https://wordpress.org/?v=4.9.7 32490438 Un petit alias bash pour git que j’aime bien http://sametmax.com/un-petit-alias-bash-pour-git-que-jaime-bien/ Sun, 16 Sep 2012 11:41:31 +0000 http://sametmax.com/?p=2167 Dans mon .bashrc:

function commit {
    git commit -m "`echo "$*" | sed -e 's/^./\U&\E/g'`"
}
alias co=commit;

Et du coup au lieu de taper:

git commit -m "Encore merci et bonne chance, nous sommes avec vous"

Je tape:

co encore merci et bonne chance, nous sommes avec vous

Notez la mise en majuscule automatique, le comble de la feignasse ;-)

Bon, c’est juste sur ma machine, car sinon co beaucoup de gens l’utilisent pour checkout.

Pour le reste, j’utilise SCM breeze bien entendu.
